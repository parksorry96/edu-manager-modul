# 개발 환경 정보

## 개발 도구
- **IDE**: IntelliJ IDEA
- **OS**: macOS (MacBook)

## 기술 스택
- **Backend**: Java 21 + Spring Boot 3.5.3
- **Frontend**: React + Vite + MUI + pnpm
- **Database**: PostgreSQL + MongoDB + Redis
- **Messaging**: Apache Kafka
- **Additional**: MapStruct, Swagger, JWT, ESLint, Prettier

## 프로젝트 구조
- 멀티모듈 Gradle 프로젝트
- 도메인 중심 설계 (DDD)
- 마이크로서비스 지향 아키텍처
